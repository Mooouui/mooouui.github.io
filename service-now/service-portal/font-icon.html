Fonts & Icons

在 Portal 添加字体

  1. 通过 attachment 方式或者 Scripted Rest API 方式引入文字资源

available font types you could use:

TypeExtension
TrueType.ttf
OpenType.otf
The Web Open Font Format.woff
The Web Open Font Format.woff2
SVG.svg
Embedded OpenType Fonts.eot
  1. 使用
/* src 可以用attachment 或者api 的 url */
@font-face {
  font-family: "example";
  src: url("/sys_attachment.do?sys_id=<attached_file_id>");
  font-weight: normal;
  font-style: normal;
}

div {
  font-family: "example";
}

TIP

直接使用 attachment.do 的方式可能会有读取权限的问题，一些 user 可能没有权限。 如果是用 public 的 api 再由 api 去处理读取值，则应当不存在此问题

在 Portal 引入 Iconfont(symbol 形式)

  • 引入 js 库(通过 dependencies)

Sample:

<template>
  <svg class="icon" aria-hidden="true">
    <use xlink:href="#icon-example"></use>
  </svg>

  <!-- 注意： 动态绑定需要用下面方式 -->
  <svg class="icon" aria-hidden="true">
    <use xlink:href="" ng-attr-xlink:href="{{data.item}}"></use>
  </svg>
</template>
