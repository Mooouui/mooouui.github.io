Multi-row Variable Set

通过 g_form 设置 Multi-row Variable Set 的值

multiple variable set 设值

const array = {};
g_form.setValue("vs_variable_set", JSON.stringify(array)); //, 设置的是 一个 json 的 字符串。

在 Multi-row Variable Set 里面拿到外面 Variable 的值

//The g_service_catalog_api has only one method available (or one documented, there might be more, who knows), method "getValue()" which can be used like:
g_service_catalog.parent.getValue("your_variable");

读取 multi-row variable set 的 行 数据

current.variables.mtr_vs_business_card_information.getRow(i); //
current.variables.mtr_vs_business_card_information.getRowCount(); //

g_form 读取 variable set 值

//We can access the MRVS values via getValue, using the "IO:" prefix, followed by the MRVS sys_id
var mrvs = g_form.getValue("IO:46c07bd31b25041064d6337bcd4bcb26");

Catalog & Catalog Item

Catalog Client Script 获取 URL

function onLoad() {
  var url = top.location.href; // get the url

  var regId = new URLSearchParams(url).get("reg_id");

  var dirId = new URLSearchParams(url).get("dir_id");

  var instName = new URLSearchParams(url).get("inst_name");

  var patchName = new URLSearchParams(url).get("patch_name");

  var type = new URLSearchParams(url).get("type");

  g_form.setValue("reg id variable", regId); // use here proper variable name

  g_form.setValue("dir id variable", dirId); // use here proper variable name

  g_form.setValue("instName variable", instName); // use here proper variable name

  g_form.setValue("patchName variable", patchName); // use here proper variable name

  g_form.setValue("patchNtypeame variable", type); // use here proper variable name
}

Catalog Client Script OnLoad 事件

  • isLoading:在 onload 和 onchange 都会生效的 script 可以去掉 isloading 只写 onchange ， 去掉 isloading 后 默认 onload 时也会执行 onchange 方法
  • select box type: variables select box type 在没有 默认值的情况下，如果 client script 的 onchange type 不要 isloading 默认 会在 loading 的时候去取一次，此时 select box type 的 variable 会默认设置 值 为 choice 的第一个值

  • 没有设置 category 的 catalog item 不能通过 search source 搜出来
  • catalog item table 有一个 column hide on service portal ， 设为 true 会在 portal 隐藏该 catalog item

function getVariablesData(fields) {
  var reqData = {};
  for (var obj in fields) reqData[fields[obj].name] = fields[obj].value;
  return reqData;
}
